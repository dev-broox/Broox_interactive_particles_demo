# Broox Interactive Particles Demo
Interactive web application that transforms text into dynamic particle visualizations, integrating with Broox's Vision Node—advanced computer vision edge software—where particles react intuitively to hand movements for an engaging, real-time interactive experience.

This demo is based on [Mamboleoo](https://codepen.io/Mamboleoo/pen/obWGYr).

## Getting started

Run `npm install` to install dependencies.
Run `npm start` to run the application on a web server. Navigate to `http://localhost:1234/`.

## Settings

Set prefered word and colors in dist/config.json

## Skeleton Settings

During execution, press key "s" to edit the settings. Update hand parameters in order to modify the force applied to word's particles.
